## Project Overview

The MultiModal AI Chatbot is a Streamlit-based AI application built using Google Gemini AI.
It allows users to chat with an AI assistant, upload images for analysis, and perform multimodal interaction using both text and images.

It combines semantic search with Large Language Models to provide accurate and context-aware responses.

---

## How It Works

- User enters a text prompt
- Gemini AI processes the query
- The chatbot generates an AI response
- Users can upload images for analysis
- Gemini Vision analyzes the uploaded image
- he AI generates contextual responses using both text and image inputs
