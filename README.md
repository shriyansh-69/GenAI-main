# GenAI

This repository contains Generative AI projects focused on building intelligent chatbots and AI applications.  
It covers core technologies such as Large Language Models (LLMs), Vector Databases, APIs, and Retrieval-Augmented Generation (RAG).  
The projects demonstrate real-world use cases, integrations, and best practices for modern GenAI systems.
Each project in this repository is maintained in its own dedicated folder for better organization and scalability.
